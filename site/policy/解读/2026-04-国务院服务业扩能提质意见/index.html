    国务院《关于推进服务业扩能提质的意见》· 政策解读¶

    发文机关：国务院
    印发日期：2026-04-14
    对外发布：2026-04-21
    解读日期：2026-07-07
    来源等级：T1（国务院正式印发）
    核心意义：首次国家级正式文件将"大模型"和"智能体服务"纳入采购支持清单，标志中国 AI 产业从"鼓励探索"迈入"制度性采购"新阶段
    联动来源：2026-07-07 从 市场监测/周报/2026-07-07-真实数据周报.md 发现

    一、政策基本信息¶

    项目 内容
    文件全称 《国务院关于推进服务业扩能提质的意见》
    发文机关 国务院
    印发日期 2026-04-14
    对外发布 2026-04-21
    政策层级 国务院层面综合性行动意见（高于部委意见）
    覆盖范围 全国服务业
    核心定位 服务业高质量发展路线图；到 2030 年服务业总规模达 100 万亿元
    对 AI 产业的历史性意义 首次国家级正式文件明确"支持采购大模型、智能体服务"

    来源：证券时报 · 国务院原文 · 北京日报 · 全文 · 腾讯新闻 · 深度解读 · 新浪财经

    二、核心内容摘要¶

    与 AI 相关的关键条款¶

    1. 软件领域：深入实施"人工智能+"行动，加快智能编程工具研发使用，支持采购大模型、智能体服务
    2. 人力资源领域：开发招聘大模型和虚拟现实培训服务产品，实施专业技术人才知识更新工程
    3. 教育相关领域（虽本意见未专门列教育章节，但通过"人力资源"和"服务业"辐射）：教师培训、教研服务、教育资源服务均在支持范围内
    4. 配套支持：下一代通信技术、数据要素市场、金融工具配套形成完整赋能闭环
    5. 总体目标：到 2030 年服务业总规模达 100 万亿元，培育"中国服务"品牌

    关键判断标准¶

    "支持采购"四个字的政策含义：

    • 相当于从"你可以试试看"变成"我帮你买单"
    • 各级政府、国企、事业单位：AI 服务从"可有可无的增量投入"→ 可以纳入常规采购清单的标准化服务
    • 政府采购目录中新增 AI Agent、大模型服务品类的直接政策依据

    三、对学科网的影响¶

    3.1 直接产品线机会¶

    产品 影响等级 说明
    AI 小博士（AI 备课 Agent 形态） 🟢 重大利好 Agent 化产品进入政府采购目录的直接政策依据；如学校/教育局/事业单位启动 AI 备课工具采购，AI 小博士是合规候选
    AI 研修（教师成长 Agent） 🟢 重大利好 与"人力资源领域 · 专业技术人才知识更新工程"完全对齐；教师培训类采购的直接对接产品
    e卷通 / 组卷网 🟡 中等利好 大模型能力增强的智能评卷/组卷可以对接采购目录
    网校通（B 端 SaaS 平台） 🟡 中等利好 平台承接大模型/Agent 服务的载体
    开放平台 🟡 中等利好 大模型 API 服务的对接通道

    3.2 关联政策矩阵形成的"双支持"¶

    《"人工智能+教育"行动计划》（2026-04，教育部教科信〔2026〕1号）
   ↓ AI+教育应用方向
《国务院服务业扩能提质意见》（2026-04-14）
   ↓ 支持采购大模型/智能体
《智能体规范应用与创新发展实施意见》（2026-06-14，三部门）
   ↓ Agent 备案要求

    关键判断：国务院 4-14 意见与教育部教科信 1 号形成"国务院支持采购 + 教育部推动应用"的双政策矩阵，为 AI+教育产品打开了完整的政府采购通道。

    3.3 时间窗口¶

    • 短期（3 个月）：政府采购目录调整；国企/事业单位的 AI 采购目录制定
    • 中期（6-12 个月）：各省承接细则出台；采购规模化释放
    • 长期（1-2 年）：AI Agent 服务成为教育信息化采购的标准品类

    四、资金流向判断¶

    • 服务业 100 万亿元目标：到 2030 年，教育服务业作为服务业重要组成部分，将获得配套增量投入
    • 政府/国企/事业单位 AI 采购预算：估算 2026-2027 年全国 AI Agent 服务采购规模 100-300 亿元（保守估计），其中教育领域占比 15-25%
    • 对学科网：如果 AI 小博士、AI 研修、e卷通全线打入政府采购目录，Q4-2027 年潜在采购机会 5-15 亿元规模

    五、建议行动¶

    受众 行动
    产品线 ①立即启动 AI 小博士、AI 研修的智能体备案（对齐 6-14 三部门实施意见）；②推动 AI Agent 化产品符合政府采购目录标准（性能指标、安全性、合规性）；③e卷通评估"AI 智能评卷+人工复核"服务化对齐国务院意见
    市场线 ①销售话术卡增补"国务院首次支持采购大模型、智能体服务"专项段落；②启动政府采购目录申请（各省/直辖市/央企/事业单位）；③2026-Q3 政府采购目录调整期为主动申请窗口
    决策层 ①将 AI Agent 化确定为下一代战略核心（3 份政策合力：教科信 1 号 + 4-14 国务院意见 + 6-14 三部门 Agent 实施意见）；②评估设立"政府/国企/事业单位 B 端"专项团队；③预算增列 AI Agent 服务的产品化+备案+合规投入

    六、与其他政策的交叉影响¶

    已有/关联政策 与本意见的关系 叠加效应
    《教育强国建设规划纲要（2024—2035）》 教育强国目标下的服务业配套 为教育数字化提供五年财政承诺
    《"人工智能+教育"行动计划》（2026-04，教科信〔2026〕1号） 教育部 AI+ 应用与国务院采购支持形成矩阵 AI+教育产品既有政策方向又有采购通道
    《智能体规范应用与创新发展实施意见》（2026-06-14） Agent 备案是采购的前置合规条件 备案 → 进入采购目录 → 获得政府采购
    《拟人化互动办法》（2026-04-10） 学习教育服务豁免 → 不构成采购障碍 学科网工具型产品无合规争议
    十五五规划（2026-06-11）教辅前置审核条款 AI+SaaS 不属于教辅材料 → 避开审核成本 学科网数字化产品保留成本优势

    七、后续追踪¶

    • 2026-Q3：观察政府采购目录调整
    • 2026-Q3-Q4：各省是否出台"AI Agent 采购目录"配套文件
    • 2026-Q4：竞品是否已进入政府采购目录
    • 2027-Q1：首批教育信息化项目中"教育智能体"品类招标出现
