    猿辅导-飞象老师 | 深度分析报告（2.0版本专项）¶

    分析日期：2026-05-27
    分析范围：飞象老师2.0（2026年5月26日发布）版本升级全维度深度分析
    信息来源：飞象老师官方网站、芥末堆、多知网、腾讯新闻、搜狐等多家媒体报道

    概况¶

    • 产品名称：飞象老师
    • 所属公司：猿力科技（猿辅导集团）/ 北京飞象星球科技有限公司
    • URL：feixianglaoshi.com
    • 产品类型：🔴 核心直接竞品（AI备课赛道）
    • 当前版本：2.0（2026年5月26日发布）
    • 上线时间：2025年12月1日（1.0版本）
    • 标语："一份教案生成一整节互动课"

    一、产品功能分析¶

    1.0 → 2.0 核心变化对比¶

    维度 1.0 版本 2.0 版本
    核心能力 "一句话"生成教学动画 上传一份教案，生成整节互动课
    产品定位 单点资源生成工具 课堂运行系统（组织教学流程）
    课件形态 独立动画/课件片段 完整HTML交互课件（会动、能练、有反馈）
    课堂数据 不支持 实时数据回收 + 教师数据看板
    教案输入 仅文字/语音 上传教案文档 → AI理解课标教材后生成
    教学闭环 碎片化 备课-授课-互动-测评 完整闭环

    2.0 新增核心功能¶

    (1) AI 互动课件生成（最核心升级）¶

    核心流程：

    上传教案/输入教学构想/拍照课本 
→ AI校准课标教材版本 
→ 理解教学目标、重难点、学生基础
→ 生成课件大纲（教师确认） 
→ 生成完整互动课件
→ 可视化讲解 + 游戏化互动 + 数据回收

    可配置选项：

    • 年级、册次、课时内容、教学重点
    • 授课类型、课件页数、班级学情

    课件内容四大模块：
    | 模块 | 说明 |
    |------|------|
    | 可视化知识讲解 | 3D动画、动态演示、参数可调 |
    | 游戏化互动练习 | 点选/拖拽/分类/匹配/闯关/分组PK等 |
    | 实时课堂反馈 | 数据回收看板 |
    | 课后练习 | 智能命题、组题、批改 |

    (2) 课堂数据回收系统（最大惊喜）¶

    • 老师可在课件中直接发起练习/数据回收
    • 系统自动生成学生作答页 + 教师数据看板
    • 实时展示：
      • 学生提交情况（谁交了、谁没交）
      • 题目正确率
      • 高频错题
      • 个人作答记录
    • 分发方式：通过微信/钉钉发送链接，无需学生平板
    • 意义：课堂反馈从课后移至当堂，教学决策从"凭经验"走向"凭数据"

    (3) 3D 动态演示增强¶

    • 数学公式可直接生成3D动态推导演示（如圆柱体体积公式：切分、展开、重组）
    • 老师可调节半径、高度等参数，图形随之变化
    • 覆盖物理、化学、生物等学科的实验动画与微观模拟

    (4) 游戏化互动练习升级¶

    新增互动形式：

    • 点选、拖拽、分类、匹配
    • 闯关挑战、分组PK
    • RPG角色扮演（如"模拟人生"式剧情互动）
    • 消消乐、贪吃蛇等小游戏
    • 翻翻卡、知识擂台赛、情景探究

    (5) 多模态输入增强¶

    • 文字输入、语音输入
    • 拍照生成（拍课本页面或题目自动识别生成）

    现有功能完整盘点¶

    功能模块 功能点 说明
    🎯 备课工具 AI教学动画 一句话生成专业级交互教学动画
    AI互动课件 2.0新增 — 上传教案生成完整课件
    AI教案·大单元 自动生成符合新课标的结构化教案
    AI命题组题 分层习题/智能组卷
    AI作文批改 智能批改与反馈（测试中）
    AI错因诊断 上传错题图片，分析错因并组针对性练习
    🏫 课堂工具 数据回收看板 2.0新增 — 实时课堂数据回收
    分组PK 学生分组竞赛答题
    游戏化互动 闯关、消消乐、RPG等
    双师课堂 屏幕共享、PC端课程创建、激光笔同步
    🌐 社区 创意社区 数百万份优质资源
    一键仿制 复制他人作品快速改编

    命题Word下载 试卷下载为Word文档

    二、学科覆盖与资源分析¶

    学科覆盖范围¶

    学段 学科
    小学 语文、数学、英语、科学、信息科技
    初中 语文、数学、英语、物理、化学、生物、地理、历史
    其他 学前教育、体育（火柴人动作演示）、职业教育等

    平台资源规模¶

    • 资源总量：社区已形成"数百万份优质资源"（官方口径），实际活跃课件资源估计在数十万量级
    • 案例模板：超过3万案例模板（创意社区数据）
    • 服务规模：
      • 服务学校：3500+所（截至2026年3月）
      • 服务教师：10万+教师
      • AI通识课试点：1300+所学校，超15万学生

    各学科资源分布（估算，基于公开信息）¶

    注：飞象老师平台未公开各学科精确资源数量。以下为基于公开报道、社区活跃度的推断：

    学科 资源占比（估） 热门度 说明
    小学数学 ~30% 🔥🔥🔥🔥🔥 资源最多的学科，3D几何推导、函数图像等最受欢迎

    三、用户体验与交互流程¶

    核心任务路径（备课流程）¶

    1.0 版本（"一句话"模式）：

    输入教学想法 → AI生成单个资源（动画/练习/题目） → 教师使用

    2.0 版本（完整教案模式）：

    上传教案/输入教学构想/拍照课本 
→ 系统校准课标教材版本 
→ 理解教学目标、重难点和学情 
→ 生成课件大纲 → 教师确认/调整大纲 
→ AI生成完整互动课件（含讲解+互动+练习+数据回收） 
→ 教师逐页预览/调整 
→ 课堂授课（发起互动→回收数据→即时反馈）

    体验亮点¶

    • 先理解再生成：不是直接堆页面，而是先拆解教学目标为课堂流程
    • 教师主导：可逐页修改，保留个人讲法，AI不替代老师上课
    • 互动细节打磨：输入框打字跳动反馈、提交按钮按下反馈、素材匹配学段审美
    • 零门槛分发：无需学生平板，微信/钉钉链接即可作答

    体验痛点¶

    • 积分消耗较快（每日30积分约可生成10次），免费额度有限
    • 生成内容仅限平台内使用，无法完全脱离平台
    • 深度思考能力外界评价不如DeepSeek等通用大模型

    技术亮点¶

    • 课标理解能力：能自动校准课程标准和教材版本，理解教学目标与重难点
    • 教学流程编排：将知识点转化为课堂环节，非简单页面堆砌
    • 3D引擎内置：动态可视化能力直接嵌入课件生成流程
    • 数据回收架构：学生端无需额外App，通过链接即可完成数据回传

    • 深度推理能力外界评价弱于DeepSeek等通用大模型
    • 离线功能有限（生成课件的HTML文件可离线用，但生成过程需在线）

    六、与学科网的差异分析¶

    维度 飞象老师 学科网
    核心模式 AI生成（生成式） 内容平台（资源库模式）
    课件形态 动态HTML交互课件 静态PPT/Word/PDF
    备课效率 几分钟生成整节课 需要手动搜索+组合
    课堂数据 支持实时回收 不支持
    游戏化互动 原生内置 需第三方工具配合
    教师主导权 可逐页修改，保留讲法 直接用或二次编辑
    资源积累 社区数百万份（新兴） 千万级资源（十余年积累）
    定价 免费+积分制 会员订阅制
    技术壁垒 自研元力大模型，百亿研发 资源量与编审体系

    核心竞争态势变化（2.0版本后）：

    • 飞象老师从"工具"切入"课堂运行系统"，直接在授课场景与学科网形成竞争
    • 数据回收功能是学科网目前完全不具备的能力
    • 但学科网资源深度和编审质量仍是飞象老师短期内难以超越的壁垒

    七、总结与建议¶

    2.0升级判断：🚨 重大¶

    飞象老师2.0不是一次常规更新，而是产品形态的质变：

    1. 从"单点工具"到"课堂系统" — 这是战略级升级，直接进入课堂核心环节
    2. 数据回收是杀手级功能 — 把课堂反馈从课后移至当堂，对传统备课平台是降维打击
    3. AI理解课标+流程编排 — 技术门槛极高，非简单套壳AI可比

    需关注的动态¶

    • 作业批改功能测试中（AI作文批改+错因诊断），后续可能构成更完整闭环
    • 2.0版本是否会推动学校/教育局整体采购（目前以个人使用为主）
    • 猿力科技后续是否会将数据回收能力与飞象星球整体方案打通

    建议行动¶

    1. 🔴 尽快对标体验 — 亲自使用飞象老师2.0生成一次完整课件，亲身体验"教案→互动课"的全流程
    2. 🔴 关注数据回收功能 — 这是最直接的差距点，需评估是否要在学科网产品中规划类似能力
    3. 🟡 跟踪用户迁移 — 飞象老师2.0可能会吸引原本使用传统资源平台的教师，需监测用户流失情况
    4. 🟡 研究3D互动课件技术栈 — 飞象老师的HTML互动课件技术路线值得借鉴
