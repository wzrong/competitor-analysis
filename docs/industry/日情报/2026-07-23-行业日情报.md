## 今日情报

### 【行业融资】2026上半年中国教育融资创近六年最低，AI项目占43.8%但规模偏小

- 事件：据创事记7月22日发布的H1融资报告，2026年上半年中国教育行业共发生**16起**融资事件，可统计金额**1.21亿元**，较2025年上半年同比减少**33.3%**（事件数）和**89.6%**（金额），均为近六年最低；天使轮占**81.3%**；项目名称或业务描述中明确出现"AI"的项目**7起（占43.8%）**，横跨职业教育、教育科技、教育企业服务和早幼教，但均为百万至千万级小额早期项目。职业教育以7起事件（金额4800万元）居细分赛道首位。[T2-行业报告/媒体, 创事记/新浪科技, 2026-07-22]
- 来源：[2026上半年中国教育行业融资风向报告，创投圈的钱都去哪了？](https://finance.sina.com.cn/tech/csj/2026-07-22/doc-iniispmy8723351.shtml)
- 对学科网的影响：
    - 信息：一级市场融资低迷意味着新入场小型竞品资金来源趋紧，头部成熟平台的相对优势窗口扩大
    - 机会：融资萎缩期具备资金储备的成熟平台有并购整合早期AI工具标的的机会窗口
    - 信息：AI嵌入度升高（占43.8%）是结构性趋势，验证AI工具化方向判断正确；但这些早期项目均未形成规模，暂不构成直接竞争威胁

### 【基座能力】Kimi K3引发"开源炸榜"叙事持续，教育AI基座能力传导效应延续

- 事件：月之暗面7月17日在WAIC发布**Kimi K3**（**2.8万亿参数**，全球最大规模开源模型），7月22日中华网等媒体持续报道该模型"引爆行业震动"；Kimi K3在Frontend Code Arena评测中以**1679分**超越Claude Fable 5和GPT-5.6 Sol排名第一，完整权重于7月27日前发布。该模型原生支持视觉理解，覆盖软件工程、知识工作、深度研究、多模态理解等场景。[T2-行业报告/媒体, 中华网/新华网, 2026-07-22]
- 来源：[Kimi开源模型引爆行业震动](https://news.china.com/socialgd/10000169/20260722/49626481.html) · [新华网：中国企业发布全球最大规模的开源模型Kimi K3](http://www.xinhuanet.com/tech/20260717/da893d3a5e1b429ea79d928e02847744/c.html)

## 本周值得关注

- **教育融资持续低迷，AI仍是唯一获资热词**：H1 16起事件均为早期小额，说明AI教育产品尚未进入成长轮扩张阶段；资本正在等待场景付费闭环被验证，这是行业整体处于"早期验证期"的信号
- **基座能力"开源化"加速**：Kimi K3、DeepSeek V4、腾讯混元3前后发布，教育AI产品可低成本接入高质量基座，技术壁垒快速收窄，内容/教研/渠道成为核心护城河
- **WAIC后效应收尾**：本周进入WAIC深度解读与落地验证阶段，炸榜与发布之后需观察产品实际部署节奏，教育AI赛道进入"有没有真实付费场景"的检验期
