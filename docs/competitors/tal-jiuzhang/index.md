**商业模式画像**

- **核心收入来源**：C端学员服务（主）+ B端教师工具（九章爱学）+ C端学生工具（小精龙）；教师工具当前Token近成本价，在拉规模阶段
- **客户获取逻辑**：免费积分送2000积分（拉新）→ 习惯养成后Token付费（转化）；好未来品牌背书降低获客成本
- **增长飞轮**：170+智能体生态→功能覆盖全周期→教师日常离不开→社区口碑传播→更多教师加入→智能体越来越丰富
- **防御壁垒**：九章大模型（MathGPT）数学专项能力（技术壁垒）+ 好未来30年教育研发积累 + 170+智能体生态的使用惯性
- **当前战略重心**：从「AI工具」向「AI老师智能体（L3级别）」升级，商业模式从「自愿付费」转向「使用即付费（Token购买）」——规模优先，变现后置

> **解读动作时的参照**：好未来的每个产品动作都在验证「AI老师L3→L4→L5」的技术路线——凡是推动AI从辅助工具变为能独立完成多步骤教学任务的功能，都是战略级推进；积分/Token定价变化是商业化节奏的重要信号。

## L2 产品纵深

**产品矩阵**：

- **九章爱学（老师版）**：教师专属AI教学助手，覆盖备课→授课→评测→家校全周期
- **九章龙虾（桌面智能体）**：本地沙箱运行，符合校园安全要求，AI能力达L3
- **小精龙（学生端）**：2026-04-01上线，基于OpenClaw架构，原生学生端精准学产品

**核心功能（老师版）**：

- 一句话生成完整课件（PPT）
- 试卷诊断报告（上传→视觉理解→批改→诊断）
- 几何讲解动画自动生成（数学专项）
- 教案撰写（覆盖教学重难点）
- 学情分析与个性化作业
- 定时任务（规律性事务自动处理）
- 170+智能体覆盖教学全周期
- **「1平台+3块屏」全场景方案**：公开展示的全场景 AI 智慧教育方案，教室大屏+教师小屏+学生Pad，打通"备-教-练-测-评"全链路

**技术架构**：

- 自研九章大模型（MathGPT）——数学领域专项优势
- 混合模型架构（教育场景用自研，日常用通用大模型）
- 基于OpenClaw架构深度定制（非直接使用开源框架）
- 本地沙箱运行，校园数据安全合规
- CTO田密提出AI老师L1-L5分级理论，九章龙虾已达L3水平

## L3 价格力

- **定价模式**：积分制（按Token消耗计费），下载即送2000积分，当前定价基本只覆盖成本；App Store 显示九章爱学（老师版）免费下载且存在 App 内购买入口，公开内购项为会员连续包月 ¥30、连续包季 ¥88、连续包年 ¥328；另有 App Store 老师版独立页标题为「九章爱学老师版|一键生成教案课件AI助手」，可作为老师版 App 入口证据，但仍不能替代 Token 包价格
- **商业化节奏**：「自愿付费」→「使用即付费（Token购买）」——规模优先，变现后置
- **Token包价格**：⚠️待核实（App Store 已显示会员内购价格，但未公开具体 Token 包价格，不能用会员价格替代 Token 包价格）
- **B端学校定价**：「1平台+3块屏」整体方案，学校集采价格⚠️待核实
- **与学科网价格位置**：定价模式截然不同（Token vs 年费）；当前阶段低价策略是战略选择而非产品定价能力不足

## L4 渠道力

- **C端渠道**：好未来品牌背书（30亿美元体量）+ 教师口碑传播 + 积分免费策略拉新
- **B端渠道**：好未来原有学校关系 + 世数会亮相获教育部认可 + 与重点高中战略合作（如仁北高中联合实验室）。好未来官网确认仁北高中联合实验室及九章爱学投入使用数月，可作为标杆校合作证据，但复制范围和规模化落地情况仍待核实。
- **与教育局/教研室关系**：好未来集团层面有省级政府合作基础 ⚠️具体覆盖范围待核实
- **招投标能力**：⚠️待核实（以C端工具为主，B端整体采购尚在建立）
- **硬件渠道**：「3块屏」方案明确涉及教室大屏、教师小屏、学生Pad；北京师范大学相关 PDF 提到 40 台九章爱学精准教学平板的试点设备线索，但具体供应链、硬件合作伙伴、采购单价和打包方式未公开 ⚠️待核实

## L7 战略卡位

- **主赛道**：AI备课/教学全周期（教师端）+ AI精准学（学生端，延伸）
- **竞争关系**：🔴 核心直接竞品（AI备课赛道，与飞象老师并列为学科网最强直接威胁）
- **与学科网重叠场景**：AI备课/课件生成、教案撰写、学情分析
- **好未来有/学科网弱**：数学专项AI（MathGPT）、学生端精准学（小精龙）、170+智能体生态、L3级AI智能体能力
- **学科网有/好未来弱**：内容资源体量（2700W+）、题库深度（3000W+）、教师研修体系
- **本质差异**：好未来走「AI生成+智能体」路线，学科网走「内容深度+AI辅助」路线——本质是两种教育AI路线的竞争
- **竞争强度**：🔥 激烈，好未来的资金投入能力决定竞争会持续加剧

## L8 战略研判

### 8A 动向研判

- **产品方向**：从L3向L4/L5持续迭代（AI从「能干活」到「能替代」）；小精龙学生端深化打通B/C数据闭环；全学科覆盖补强（当前数学最强，其他学科追赶）
- **市场方向**：20万教师→百万教师（量级突破）；「1平台+3块屏」B端整体采购规模化
- **商业化方向**：Token近成本→Token合理盈利（当用户规模达到临界点后快速提价）
- **信号关注**：Token定价调整（商业化转折点）；智能体数量（从170+到多少）；与硬件厂商的合作动态（「3块屏」供应链）

### 8B 管理层假设

- **他们认为的核心优势**：「AI技术深度+垂直数学能力」——CTO田密提出L1-L5分级理论，表明好未来相信自己是AI教育领域最懂技术路线的玩家
- **对学科网的认知**：可能视学科网为「内容平台型选手」，认为在AI时代内容量不再是核心护城河，算法和模型才是
- **认知偏差**：好未来的技术信仰可能让他们低估了「内容准确性」在K12教学中的刚性需求——教师宁可慢一点，也不能用错误的内容教学生；也可能低估了B端学校对平台稳定性和本地化服务的重视
- **战略盲点**：数学专项能力强但全学科覆盖需要时间；Token商业化时机选择（过早提价会流失用户，过晚影响财务）；政策风险对任何C端大规模教育业务都是悬剑
- **反攻触发条件**：如果学科网推出更强的AI备课工具或与好未来客户有重叠，好未来可能加大免费补贴或加速L4/L5能力发布
