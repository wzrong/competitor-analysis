## 主要功能

- 一句话生成完整课件（PPT）
- 试卷诊断报告（上传→视觉理解→批改→诊断）
- 几何讲解动画自动生成
- 教案撰写（覆盖教学重难点）
- 学情分析与个性化作业
- 定时任务（规律性事务自动处理）
- 集成 170+ 智能体覆盖教学全周期
- 「1平台+3块屏」全场景方案：教室大屏+教师小屏+学生Pad，打通"备-教-练-测-评"全链路

## 技术栈 / AI 能力

- 自研九章大模型（MathGPT）
- 混合模型架构（教育场景用自研，日常用通用大模型）
- 基于 OpenClaw 架构深度定制，非直接使用开源框架
- 本地沙箱运行，符合校园安全要求
- CTO田密提出AI老师L1-L5分级理论，九章龙虾达到L3水平

## 与学科网的差异

- 好未来的"Skills生态"和"积分制免费策略"——用免费拉教师规模，再通过Token变现
- "1平台+3块屏"架构覆盖课前中后全场景，是目前最完整的AI备课方案之一
- 好未来30亿美元营收体量，与学科网不在一个量级，其AI战略的资金投入能力远超学科网
- 九章龙虾达L3水平（CTO自述），意味着AI已从辅助工具变为可独立完成多步骤教学任务的智能体
- 学科网需要在AI备课的差异化（内容深度、本地化）而非通用AI能力上建立壁垒
