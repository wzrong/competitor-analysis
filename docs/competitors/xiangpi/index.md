**商业模式画像**

- **核心收入来源**：学校采购（年费制，如单校约5.6万元/年）
- **客户获取逻辑**：与正确云捆绑销售（橡皮题库+正确云备课），打包获客
- **增长飞轮**：题库+备课云捆绑→学校一次采购解决多场景→复购率高
- **防御壁垒**：与正确云协同的平台捆绑销售；独家视频资源（高考专家讲座）
- **当前战略重心**：稳固题库组卷优势，与正确云深度整合

## L2 产品纵深

**核心定位**：K12标准化题库系统——章节出题+知识点出题+智能组卷

**主要功能**：

- 章节选题 / 知识点选题 / 试卷选题 / 智能组卷 / 细目表选题
- 1500万+试题（全学段全学科全版本），每日更新1万+道
- 精品资源 / 每周好题
- 独家视频资源（高考专家讲座、名师经验分享）
- 有效功能页：`/chuti/shijuan` 试卷出题页已核验，显示去重题库总量 1522.81 万题、24h 更新 10417 题、章节/知识点/名校试卷/智能选题/细目表选题等入口 [T1-公开可查证, 官网实测, 2026-06-02]

## L3 价格力

- **定价模式**：学校年费采购（单一来源/公开招投标）
- **典型案例**：新疆师范大学附属中学橡皮网采购项目，单一来源公示，预算金额 56,000 元，拟定供应商为北京正确教育咨询有限公司 [T1-公开可查证, 学校采购公示, 2026-06-01]
- **与学科网价格位置**：B端年费制，与学科网相近；打包正确云时性价比更高

## L6 SWOT

| | 内容 | 对学科网的含义 |
|---|---|---|
| **S 优势** | 与正确云协同的打包销售能力；独家高考专家视频资源是差异化内容 | 需要持续关注此优势是否在侵蚀学科网的核心用户 |
| **W 劣势** | 题库量（1500万）少于学科网（3000万+）；独立品牌影响力有限 | 学科网的机会区间 |
| **O 机会** | AI+教育政策利好，有AI升级机会 | 若AI升级成功，竞争压力上升 |
| **T 威胁** | AI生成工具对传统资源/题库平台构成降维威胁 | 学科网面临共同外部威胁 |
