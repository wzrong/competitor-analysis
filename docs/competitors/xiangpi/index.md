## 主要功能

- 章节选题 / 知识点选题 / 试卷选题
- 智能组卷 / 细目表选题
- 精品资源 / 每周好题
- 独家视频资源（高考专家讲座、名师经验分享）

## 技术栈 / AI 能力

- 题库规模：1500万+试题，全学段全学科全版本覆盖，每日更新1万+道
- 专业教研团队人工标注知识点、章节、难度；暂无公开AI能力

## 与学科网的差异

- 橡皮网题库规模（1500万+）远小于学科网（2800万+）
- 橡皮网人工标注质量被用户认可，学科网AI化程度更高
- 正确云+橡皮网的组合，在功能上和学科网+组卷网的组合高度对应
- 体量差距明显，但在部分学校的单一来源采购中有稳固地位
