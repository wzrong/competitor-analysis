**商业模式画像**

- **核心收入来源**：B端学校智慧体育教育解决方案（SaaS+硬件）；军体训练系统（军队采购）
- **客户获取逻辑**：以智慧体育为入口切入学校，通过体育学科的备课/教学/考试/训练四场景建立客情
- **增长飞轮**：智慧体育解决方案→学校体育教学数字化→体测/训练数据积累→AI训练方案优化→更多学校采用
- **防御壁垒**：5G+智慧体育教育是工信部/教育部双部委试点项目，政策壁垒高；军体训练系统获军委验收，有独特的非教育领域收入来源
- **当前战略重心**：深耕智慧体育赛道，从学校体育教育向军队训练、康复医疗延伸

> **⚠️ 重要重新评估（2026-05-31）**：经深度搜索确认，广州中金育能的核心赛道是**智慧体育教育**，不是传统意义上的学术精准教学。展会表中"备课/学情/智慧课堂/阅卷作业"四个子场景实际上是围绕体育学科的——智慧体育备课、体育训练数据分析、智慧体育课堂、体育考试系统。与学科网的资源备课/组卷/阅卷/学情分析在学科范围上基本不重叠。**竞争关系从🟡调整为🟢（间接竞品），威胁趋势从↑上升调整为→稳定。**

## L2 产品纵深

**核心产品线（体育学科专用）**：

| 产品 | 定位 | 核心能力 |
|---|---|---|
| 育能智慧课堂系统 | 体育教学数字化 | 三段式教学设计（科学化+趣味化）；个性化教案生成；智能化备课 |
| 智慧备课系统 | 体育教师备课工具 | 体育课程设计、训练方案规划 |
| 智慧考试系统 | 体育考核/体测 | 运动数据采集、成绩评定、素质分析 |
| 个性化训练系统 | 学生个性化体育训练 | AI训练方案生成、训练效果追踪 |
| AI训考一体机 | 体育考试/训练硬件 | 软硬件一体化，体育考试场景专用设备 |
| 智慧康复平台 | 运动康复/医疗 | 运动伤害防护、康复训练方案 |
| 军体训练考核系统 | 军队体能训练 | 中央军委训练管理部验收 |

**与学科网重叠度分析**：

- 学科网无体育学科产品线——体育是学科网的空白学科
- 即使未来学科网扩展体育学科，中金育能的硬件+软件+可穿戴设备的体育教学模式也与学科网的纯软件/内容模式完全不同
- 竞争交集仅存在于"智慧课堂""备课系统"的功能名称层面，实际学科和用户场景无实质重叠

## L8 战略研判

### 8A 动向研判

中金育能在智慧体育赛道的先发优势可能持续扩大（双部委背书+军方采购），但不会对学科网构成威胁。体育教育数字化是一个独立的市场，不在学科网当前赛道范围内。

如果国家体育中考分值持续提升（目前部分省份已提到100分），智慧体育赛道的市场规模将快速扩大。但这是学科网的机会（可扩展体育学科）而不是威胁。

### 8B 管理层假设

中金育能可能相信"体育教育数字化是未被充分开发的蓝海市场"——基于双部委验收的成功经验，他们可能继续深耕体育+健康赛道而非横向扩展到其他学科。

对学科网的认知偏差：中金育能可能完全不关注学科网——因为赛道不同。学科网也不需要过度关注中金育能，除非决定进入体育学科。
