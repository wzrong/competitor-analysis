# 国家中小学智慧教育平台 | 深度分析报告

**分析日期**：2026-05-23
**分析范围**：产品功能、用户体验/交互、目标用户与定位、技术能力四维度全覆盖
**分析背景**：2026年5月11-13日世界数字教育大会期间，国家中小学智慧教育平台正式发布系列AI工具和AI通识教育资源，标志着其从"纯资源平台"正式向"AI+教育平台"转型。本次变化对学科网B端付费模式构成结构性威胁，需深度评估。

---

## 概况

- 产品名称：国家中小学智慧教育平台
- 所属公司：中华人民共和国教育部（官方公益平台）
- URL：https://www.smartedu.cn/
- 产品类型：🟡直接竞品（官方免费教育资源平台，对学科网B端付费模式构成结构性压力）

---

## 一、产品功能分析

### 核心功能

国家中小学智慧教育平台的功能可分为"基础资源层"和"新增AI层"：

**基础资源层（此前已有）：**

| 功能模块 | 实现方式 | 与学科网的重叠度 |
|---------|---------|----------------|
| 课程教学资源 | 全学段、全学科视频课程+配套资源，8.8万+条数字教育资源 | 高（学科网核心功能） |
| 专题教育 | 安全教育、心理健康、德育等专题内容 | 中（学科网有部分专题资源） |
| 教师研修 | 在线培训课程 | 中（学科网有教师培训内容） |
| 家庭教育指导 | 面向家长的教育指导资源 | 低（学科网无此功能） |
| 教材电子版 | 各学科教材电子版本 | 中（学科网有教材相关资源） |

**新增AI层（2026年5月发布）：**

| 功能模块 | 实现方式 | 与学科网的重叠度 |
|---------|---------|----------------|
| 系列AI教育工具 | 汇聚十余款优质AI工具，推动AI走进课堂 | 高（与学科网AI小博士直接竞争） |
| AI通识教育资源 | 《人工智能通识教育讲义》四册 + 《十讲百问系列课程》 | 低（学科网无AI通识教育内容） |
| AI智能搜索/视频导航 | AI视频摘要、大纲提炼、知识点分段定位 | 中（学科网搜索功能较传统） |
| AI智能助教 | 辅助教师备课授课，提供知识点解析、课程推荐 | 高（与学科网AI小博士竞争） |
| AI智能助学 | 为学生生成个性化学习计划、知识点地图、今日练习 | 高（与学科网精准教学竞争） |
| AI试验场 | "AI课堂""数学解题助手""智能出题助手"等实用工具 | 高（与学科网组卷、AI解题竞争） |
| 猿创未来AI教育平台接入 | 猿编程打造的AI教育工具平台正式接入，提供AI绘画、AI音视频、AI数字人、模型训练等 | 中（学科网无此类工具） |

### 独有功能（学科网未覆盖）

1. **官方权威性与版权安全** — 关注度：**高**
   - 作为国家平台，所有资源均经过教育部审核，无版权风险。学校使用无需担心侵权问题。
   - 这在政府采购和学校合规层面是学科网无法比拟的优势。

2. **AI通识教育体系化内容** — 关注度：**中**
   - 发布小学低年级/高年级/初中/高中四册《人工智能通识教育讲义》及配套课程。
   - 这是响应教育部"加快普及中小学生人工智能教育"政策方向的系统布局，学科网目前无对应产品。

3. **免费+政策背书** — 关注度：**高**
   - 完全免费，且带有教育部官方背书。在预算有限的学校，尤其是农村和边远地区学校，具有极强的吸引力。
   - 教育部等五部门联合印发的《"人工智能+教育"行动计划》明确支持农村、边远地区学校利用国家平台开好人工智能课程。

### 缺失功能（学科网有而国家平台未覆盖）

- **资源的深度和个性化**：国家平台资源质量标准化但缺乏深度和个性化，学科网资源由一线教师上传，更贴近实际教学需求和各地考试差异。
- **中高考专题资源**：学科网在中高考备考资源上的系统性和深度远超国家平台。
- **教师UGC生态**：学科网拥有活跃的教师上传、分享、交流社区，国家平台以官方供给为主。
- **组卷系统的灵活性**：学科网组卷系统支持极其细粒度的条件筛选和自定义，国家平台的组卷功能（如有）预计较为基础。
- **AI功能的深度**：虽然国家平台发布了AI工具，但具体功能深度（如AI解题的准确率、智能出题的灵活性）尚不明确，预计短期内难以达到商业产品的成熟度。

### 功能更新频率

- 2026年5月11-13日：世数会集中发布八项成果，AI工具和AI通识教育资源是核心新增功能
- 平台持续进行资源更新，但产品功能层面的迭代速度预计不如商业竞品敏捷

## 二、用户体验分析

### 关键流程

国家平台的核心使用流程：

**教师端：**
1. 访问 smartedu.cn
2. 按学段/学科/教材版本筛选资源
3. 下载或在线使用资源
4. （新增）使用AI智能助教辅助备课
5. （新增）使用AI试验场工具进行教学辅助

**学生端：**
1. 访问平台
2. 观看课程视频或下载学习资源
3. （新增）使用AI智能助学获取个性化学习计划
4. （新增）使用AI试验场进行练习和实验

### 体验评价

**亮点：**
- **官方权威性**：资源质量有保障，无广告干扰，使用安心。
- **全学科全覆盖**：从幼教到高中，各学科资源均有覆盖。
- **免费无门槛**：无需付费、无需复杂授权，降低使用门槛。
- **新增AI功能降低搜索成本**：AI视频摘要、大纲提炼等功能可以帮助教师快速定位资源。

**痛点/不确定性：**
- **资源深度不足**：一线教师普遍反馈国家平台的资源"有但不够深"，尤其是针对各地中高考的差异性资源匮乏。
- **更新速度偏慢**：官方资源的更新速度和商业平台相比偏慢，难以快速响应教学一线的新需求。
- **AI功能成熟度待验证**：虽然发布了AI工具，但十余款AI工具的质量、稳定性和易用性如何，需要实际使用验证。
- **个性化程度有限**：作为国家级平台，需要兼顾全国普适性，难以做到像商业平台那样针对特定地区、特定学校的深度定制。

### 可借鉴之处

1. **AI资源导航**：国家平台的AI视频摘要、知识点分段定位功能，学科网可以参考用于提升资源检索效率。
2. **通识教育内容布局**：AI通识教育是一个政策驱动的增量市场，学科网可以考虑开发相关的通识教育资源。
3. **政策合规背书**：国家平台的成功说明"官方背书"在教育市场具有极高价值，学科网可以加强与地方教育部门的合作认证。

## 三、定位与用户

- **目标用户**：全国中小学教师、学生、家长，重点是农村和边远地区学校
- **核心价值主张**："官方免费教育资源公共服务平台——全学科覆盖、无版权风险的国家级基础设施，正从资源平台向AI教育平台转型"
- **定价模式**：完全免费
- **市场切入点**：从"教育公平"和"国家基础设施"角度切入，以政策力量推动覆盖。
- **品牌调性与话术风格**：强调公益性、权威性、公平性。话术风格正式、政策导向强。
- **用户口碑**：覆盖200+国家和地区，服务1.78亿+学习者，各地密集开展平台应用培训（烟台、信阳、南京、广州等）。

### 核心数据

| 指标 | 数据 |
|------|------|
| 覆盖国家/地区 | 200+ |
| 服务学习者 | 1.78亿+ |
| 数字教育资源 | 8.8万+条 |
| 接入AI教育平台 | 猿创未来等 |

### 与我们定位的差异

| 维度 | 国家中小学智慧教育平台 | 学科网 |
|------|----------------------|--------|
| 定价 | 完全免费 | 付费订阅为主 |
| 资源深度 | 标准化、普适性 | 深度化、差异化 |
| 更新速度 | 较慢（官方审核流程） | 较快（教师UGC+编辑团队） |
| AI能力 | 刚起步（2026年5月发布） | AI小博士已上线一段时间 |
| 官方背书 | 极强（教育部主办） | 较弱（商业公司） |
| 定制化 | 无 | 可根据地区/学校需求定制 |
| UGC生态 | 无（官方供给为主） | 强（教师上传分享） |

**关键判断**：国家平台和学科网在当前的竞争关系是"免费基础版 vs 付费专业版"。对于预算充足的学校，两者可以共存（国家平台用于基础资源，学科网用于深度资源）。但对于预算紧张的学校，国家平台可能是唯一选择。随着国家平台AI功能的增强，其对学科网B端基础用户的蚕食风险将加大。

## 四、技术能力

- **已知技术栈**：
  - 此前以资源分发为主，技术栈偏向内容管理和分发平台
  - 2026年5月后引入AI能力，具体技术实现方式未完全公开
  - AI智能搜索、AI助教、AI助学等功能的具体底层模型未知（可能采用多家供应商方案）

- **数据来源**：
  - 教育部统筹的全国教材、课程标准、教学指南等官方数据
  - 各地优秀教师提供的示范课程资源
  - 接入的第三方AI教育平台（如猿创未来）提供的技术和数据支持

- **准确率/可靠性**：
  - 官方资源的准确性有保障（经过教育部审核）。
  - AI功能的准确性尚待验证。作为新上线功能，可能在初期存在不稳定或不完善的情况。

- **技术亮点/壁垒**：
  1. **国家级基础设施地位**：作为教育部主办的平台，其数据安全、内容合规、服务稳定性有国家级保障。
  2. **政策整合能力**：能够直接响应教育部政策（如《"人工智能+教育"行动计划》），快速上线政策要求的功能和内容。
  3. **国际合作网络**：世数会期间近20项国际合作集中签约，平台内容可向全球输出。

- **技术局限性**：
  - 作为官方平台，技术迭代速度可能受官僚流程制约，不如商业公司敏捷。
  - AI能力目前处于"从零到一"阶段，功能深度和用户体验与商业产品可能存在差距。
  - 缺乏教学过程数据的深度积累（不直接参与考试、作业等教学环节），在学情分析和个性化推荐上的数据基础不如智学网等商业平台。

- **API开放能力**：未明确

- **技术团队**：由教育部统筹建设和运营，技术实现可能依托外部供应商。

## 五、总结与建议

### 核心差异

国家中小学智慧教育平台与学科网最根本的差异在于：**国家平台是"官方免费基础设施"，学科网是"商业专业服务"**。两者的竞争不是同一维度的功能竞争，而是"免费是否足够好"的竞争。如果国家平台的AI功能在1-2年内达到可用水平，将对学科网的B端基础用户产生显著替代效应。

### 可借鉴点

1. **AI资源导航功能**：国家平台的AI视频摘要、大纲提炼等功能可以提升资源检索效率，学科网可以在AI小博士中引入类似能力。
2. **通识教育内容**：AI通识教育是政策明确的增量方向，学科网可以及早布局相关资源。
3. **政策合规叙事**：国家平台的成功强化了"官方背书"的价值，学科网在B端销售中可以强化自身与各地教育局的合作关系。

### 需关注的动态

1. **AI工具的实际使用效果**：国家平台汇聚的十余款AI工具，教师实际使用反馈如何？使用频率高吗？这决定了其对学科网的替代强度。
2. **猿创未来等第三方平台的接入进展**：如果更多优质的第三方AI教育平台接入国家平台，其功能丰富度将快速提升。
3. **地方教育局的采购决策变化**：在已有国家免费平台的情况下，地方教育局是否还愿意为学科网等商业平台付费？需要关注B端客户的续约和预算变化。
4. **AI通识教育的推广力度**：教育部五部门行动计划要求"开齐开足开好AI相关课程"，如果国家平台成为AI通识教育的主要载体，学科网是否也能参与这一市场？
5. **农村和边远地区市场的变化**：国家平台明确支持农村、边远地区学校利用平台开好AI课程，这些地区此前可能是学科网的增量市场，现在可能被国家平台覆盖。
