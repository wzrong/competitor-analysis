# 国家中小学智慧教育平台 档案

## 基本信息

| 项目 | 内容 |
|---|---|
| 产品名称 | 国家中小学智慧教育平台 |
| 所属公司 | 中华人民共和国教育部（官方公益平台） |
| 产品类型 | 🟡 直接竞品（官方免费教育资源平台，对学科网B端付费模式构成结构性压力） |
| 目标用户 | 全国中小学教师、学生、家长 |
| 核心定位 | 官方免费教育资源公共服务平台——全学科覆盖、无版权风险的国家级基础设施，正从资源平台向AI教育平台转型 |
| 定价模式 | 完全免费 |
| URL | [smartedu.cn](https://www.smartedu.cn/) |
| 最后更新 | 2026-05-23 |

## 主要功能

- 课程教学资源（全学段、全学科视频课程+配套资源）
- 8.8万+条数字教育资源
- 专题教育（安全教育、心理健康、德育等）
- 教师研修（在线培训课程）
- 家庭教育指导
- 教材电子版
- **【新增】系列AI教育工具**（2026年5月发布，汇聚十余款优质AI工具）
- **【新增】中小学人工智能通识教育系列资源**（四套讲义+十讲百问课程）

## 技术栈 / AI 能力

此前以资源分发为主；2026年5月世界数字教育大会后正式引入AI工具，AI能力从零到一。

## 与学科网的差异

- 国家平台完全免费，学科网以付费订阅为主
- 国家平台内容质量标准化但缺乏深度和个性化，学科网资源更丰富且深度更佳
- 🚨 **关键变化**：国家平台于2026年5月正式发布AI工具和AI通识教育资源，标志着从纯资源平台向"AI+教育"平台转型——虽当前AI功能深度有限，但趋势明确
- 长期需要关注：若国家平台持续加强AI内容和功能，可能蚕食学科网的B端基础用户
