**商业模式画像**

- **核心收入来源**：硬件销售（智慧黑板/平板/录播）+ 政府/学校采购项目
- **客户获取逻辑**：教育信息化硬件招投标 → 学校采购 → 软件/服务增值
- **增长飞轮**：硬件装机量 → 软件平台使用 → 数据积累 → 续购/升级
- **防御壁垒**：硬件市占率壁垒；国资背景的渠道优势
- **当前战略重心**：国资入主后渠道策略调整；DeepSeek接入提升AI能力

## L2 产品纵深

**核心定位**：教育信息化硬件+解决方案提供商。

**主要产品**：
- 智慧黑板/交互智能平板（市占率约28.7%）
- 智π智慧教学平台
- 鲁班云职教系统
- AI超融合教室
- 裸眼3D虚拟仿真
- 智慧体育方案
- 录播系统/激光投影

**技术栈**：已接入DeepSeek开源大模型，应用于AI课堂分析；AI能力弱于希沃，以硬件集成为主。

## L3 价格力

- **定价模式**：硬件销售+政府/学校采购项目 ⚠️具体价格待核实

## L4 渠道力

- **国资背景**：安徽国资入主后，政府渠道可能进一步增强
- **软件渠道**：⚠️待核实（软件/内容生态较弱）
