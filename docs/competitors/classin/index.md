## L2 产品纵深

**核心定位**：全球教与学一体化平台——以在线互动教室为核心，延伸至LMS+AI+平台交易。

**主要功能**：
- 在线互动教室（1对1、班课、万人直播）
- LMS学习管理系统（课前/课中/课后全流程）
- AI超级课程：AI作业批改（准确率98%）、AI学情分析、AI助教
- AgentIn智能体商店（1000+智能体）
- ClassIn小店（0佣金平台交易闭环）
- NOBOOK虚拟实验（2021年收购，强化理科场景）
- 三层记忆架构（工作记忆/情节记忆/长期记忆）
