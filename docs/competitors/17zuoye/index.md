## L2 产品纵深

**核心定位**：智慧作业驱动的区校数字化转型方案。

**主要功能**：
- 智慧作业（智慧笔+点阵纸，不改变纸质书写习惯即可采集数据）
- 学情数据看板（班级/年级/校级）
- AI智能体"17同学"：AI错题本、学情分析、AI规划师、智能学伴
- C端"一起爱学"：AI会员订阅（Q4环比增长94.6%）
- 区域教育大数据平台

**技术栈**：生成式AI智能体；智慧笔+点阵纸硬件技术。
