# dokie | 深度分析报告

**分析日期**：2026-05-23
**分析范围**：产品功能、用户体验/交互、目标用户与定位、技术能力四维度全覆盖
**分析背景**：dokie是一款基于Claude Opus 4.6的AI课件生成工具，总部位于新加坡，2026年3月上线，被称为"最强AI PPT工具"。v1.5即将支持国产操作系统和满足中国AI合规要求。

---

## 概况

- 产品名称：Dokie AI
- 所属公司：Dokie（总部位于新加坡）
- URL：https://dokie.ai
- 产品类型：🟡直接竞品（AI课件生成工具）

---

## 一、产品功能分析

### 核心功能

dokie定位为"AI驱动的演示文稿制作平台"：

| 功能模块 | 实现方式 | 与学科网的重叠度 |
|---------|---------|----------------|
| 多格式课件生成 | PDF/Word/Excel/图片/链接 → PPT | 高（与学科网课件场景竞争） |
| 主题/大纲生成完整课件 | 输入主题→生成结构化课件 | 高（与AI小博士竞争） |
| 内容优先设计 | 根据教学内容自动调整版式 | 中 |
| 数据与逻辑可视化 | 复杂公式/数据→图表 | 中 |
| AI图像生成与搜索 | 即时匹配视觉素材 | 低 |
| 聊天式修改 | 自然语言描述即可调整 | 中 |
| 直接编辑 | 网页端手动修改 | 中 |
| 自定义模板 | 需付费 | 低 |

### 独有功能（学科网未覆盖）

1. **Claude Opus 4.6驱动** — 关注度：**中**
   - 采用顶尖通用大模型Claude Opus 4.6，内容理解和生成能力可能强于教育专用模型。

2. **多格式输入** — 关注度：**中**
   - 支持PDF/Word/Excel/图片/链接等多种格式输入生成课件。
   - 学科网AI小博士目前仅支持文字输入。

3. **内容优先设计** — 关注度：**中**
   - 根据教学内容自动调整版式，而非套用固定模板。
   - 可能生成更贴合教学内容的课件设计。

4. **v1.5国产系统支持** — 关注度：**高**
   - 即将支持HarmonyOS NEXT/Deepin/UOS，满足中国AI合规要求。
   - 这意味着dokie正在积极进入中国市场。

### 缺失功能（学科网有而dokie未覆盖）

- **教育资源库**：dokie是通用课件工具，无K12教育资源库。
- **组卷系统**：无题库和组卷功能。
- **B端学校服务**：偏C端个人使用。
- **中高考专题资源**：完全不涉及。
- **教师社区**：无教师交流社区。

### 功能更新频率

- 2026年3月20日正式上线
- 2026年5月中旬：v1.5 "Summit"即将发布
- 迭代速度快

## 二、用户体验分析

### 关键流程

**课件生成流程：**
1. 上传文档或输入主题
2. AI分析内容并生成大纲
3. 确认大纲后生成完整课件
4. 用自然语言描述修改需求
5. 导出PPT

### 体验评价

**亮点：**
- **多格式输入便捷**：支持多种格式输入，降低使用门槛。
- **自然语言修改**：聊天式修改让课件调整极其便捷。
- **内容优先设计**：版式根据内容自动调整，设计质量可能更高。
- **AI图像匹配**：即时生成或搜索匹配的视觉素材提升课件美观度。

**痛点：**
- **无教育资源**：纯工具属性，不提供K12教育资源。
- **教学场景不深**：通用课件工具，不针对教学场景优化。
- **B端功能缺失**：不适合学校级使用。
- **内容准确性**：AI生成内容的教学准确性需要教师校对。

### 可借鉴之处

1. **多格式输入**：学科网AI小博士可以支持文档、图片等多格式输入。
2. **聊天式修改**：自然语言调整课件是极佳的交互方式。
3. **内容优先设计**：根据教学内容自动调整版式，提升课件设计质量。

## 三、定位与用户

- **目标用户**：教师、培训师、学生、商务人士（跨领域，教育是核心场景）
- **核心价值主张**："AI驱动的演示文稿制作平台——将文档快速转化为结构化课件"
- **定价模式**：积分制（新用户500积分，每日送200积分）+ 付费订阅 $15.90/月（年付）/ $19.90/月（月付）
- **市场切入点**：从通用AI课件生成切入，教育为核心场景
- **品牌调性**：前沿AI、设计精美、国际化
- **用户口碑**：被多篇评测称为"最强AI PPT工具"

### 与我们定位的差异

| 维度 | dokie | 学科网 |
|------|--------|--------|
| 产品属性 | 通用AI课件工具 | K12教育平台 |
| 教育资源 | 无 | 海量 |
| AI模型 | Claude Opus 4.6 | 教育专用模型 |
| 教学场景 | 不深 | 深度定制 |
| B端服务 | 无 | 强 |
| 定价 | 国际定价 | 国内定价 |

**关键判断**：dokie当前对学科网威胁有限——它是通用工具，不针对K12教育场景深度优化。但若其v1.5版本进入中国市场后加强教育场景适配，可能分流部分教师用户。

## 五、总结与建议

### 核心差异

dokie是"通用AI工具型"竞品——技术能力强，但教学场景不深。当前威胁有限，但需关注其中国市场进入后的策略变化。

### 可借鉴点

1. **多格式输入**：学科网AI小博士应支持文档、图片等多格式输入。
2. **聊天式修改**：自然语言调整课件是极佳的交互方式。
3. **内容优先设计**：自动版式设计可以提升课件生成质量。

### 需关注的动态

1. **v1.5中国市场表现**：支持国产系统后是否获得中国教育用户青睐？
2. **教育场景深化**：是否会推出教育专用版本？
3. **与教育资源平台合作**：是否会与学科网等平台合作提供教育资源？
