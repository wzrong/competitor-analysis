**商业模式画像**

- **核心收入来源**：K12 教辅图书零售（高中必刷题为王牌）；数字化平台（试题网/APP）为增值服务，非独立收入
- **客户获取逻辑**：教辅图书品牌力 → 传统渠道（省代/地代/内容电商）→ 教师/学生自然渗透
- **增长飞轮**：图书销量 → 品牌认知 → 教师选用 → 更多学校采购
- **防御壁垒**：教辅品牌力（必刷题同类第一）+ 传统渠道网络 + 教研积累（1000+场/年）
- **当前战略重心**：教辅零售保守迭代；数字化投入实质性收缩（IT 从 40+ 缩至 <5 人）；探索"理想树 AI 自主学习中心"线下扩张

> **解读动作时的参照**：理想众望的动作要放回"教辅出版公司"的画像判断——凡是围绕图书销售/线下渠道的动作是核心推进，B端数字化/AI 动作是"试探"性质。

## L2 产品纵深

**核心定位**：K12 教辅图书出版公司（数字化为增值服务）

### 教辅图书（营收主力）

| 品牌/系列 | 学段 | 产品形态 | 市场地位 |
|---|---|---|---|
| **高中必刷题** | 高中 | 同步练习 | 王牌产品，同类第一，京东教辅类领先 [T2, 开卷数据] |
| 高考必刷题/必刷卷 | 高三 | 高考复习 | 分题型强化、42套、真题系列 |
| 划重点 | 初高中 | 同步讲解 | 知识梳理+重难点标注 |
| 资源库 | 初高中 | 综合资源 | 知识大全类工具书 |
| 上分专项/攻略 | 高中 | 专项突破 | 分专题提分 |
| 英语时文 | 初高中 | 阅读专项 | 时文阅读理解 |
| 衔接新高一 | 初升高 | 衔接教材 | 初高衔接 自主预习 |
| 高考分析与启示 | 高中 | 蓝皮书 | 2026年新品 [T1, 官网, 2026-01] |

### 数字平台

**试题网（stzy.com）** — 教辅图书的增值服务端

| 维度 | 试题网 | 学科网 |
|---|---|---|
| 精品题库 | 初高各科试题（量未公开，估数十万） | 近 2000 万题 |
| 智能组卷 | 章节/知识点选题、智能组卷 | 细目表组卷、防超纲选题、精品专辑 |
| 高考分析 | AI 试卷解析（真题+答案+考点） | 数据中台化分析 |
| 备课资源 | 高中同步备课（课件/教案/学案） | 超 2000 万套，小学-高中全学段 |
| VIP 会员 | 高级会员：20 份/日、100 份/周 | 75 份/日、150 份/周（免费资源） |
| 覆盖学段 | **仅初高中**（无小学） | 小学、初中、高中、中职 |
| 开放 API | ❌ 无 | ✅ 有（学校/区域对接） |

> **访谈关键补充**：题库更新主要靠老师写稿，有征稿通道，部分稿子购买而来——**非原创生产模式**，与学科网千万级原创题库存在本质差异 [T3-内部非公开信息, 离职员工访谈, 2026-06]

| 维度 | 试题网 | 学科网 |
|---|---|---|
| 试卷数量 | 数十万 | 1000万+ 套 |
| 学科覆盖 | 初 10 科 / 高 9 科 | 初 19 科 / 高 19 科 |

## L4 渠道力

- **传统渠道（主力）**：省代、地代网络深厚；线下书店、教辅门店；受平台冲击已受阻 [T3, 访谈]
- **内容电商**：京东/淘宝旗舰店，但访谈明确"电商/直播带货做得不太好" [T3, 访谈]
- **直播**：2024-06 联合央视网"高考圆桌派"是少数亮点合作 [T1, 央视网, 2024-06]
- **B 端渠道**：合作学校**不超过 10 家**；官网案例披露 27 所 [T3/T1, 访谈/官网]
- **展会渠道**：2023 年起每年参加书展 + 1 次教装展；2023-10 亮相第 82 届教装展"理想树教育云" [T1, 教装展报道]
- **资源授权**：与作业帮等合作，将内容集成到其他平台——**可能是数字内容最务实的变现路径** [T3, 访谈]

## 核心判断（供 SWOT / L7 快速引用）

1. **短期威胁：低** —— 数字化能力实质性坍缩（IT 40+→<5）+ B 端亏损 + APP 无运营
2. **中期需跟踪**：其数字化是否重启，以及资源授权模式是否与学科网 C 端合作方形成间接竞争
3. **策略窗口**：趁其无力竞争时巩固学科网 B 端壁垒；小学学段是绝对空白区（试题网无小学覆盖）
4. **不必过度反应**：竞品定位是"让教辅更好用"的增值服务，不是"替代学科网"
