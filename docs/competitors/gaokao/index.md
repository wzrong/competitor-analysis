- 高考政策解读

- 高考成绩查询
- 新高考选科指导
- 志愿填报服务（高考帮App）

## 与学科网的差异

- 高考网偏学生端（备考+志愿），学科网偏教师端（备课+组卷），直接竞争弱
- 高考网的流量价值（日均百万级PV）远大于内容深度价值
- 关注原因：好未来的全学段生态整合能力——若高考网向教师资源方向延伸，可能产生竞争
- 在"高考备考资料"这一细分场景，高考网和学科网有局部重叠
