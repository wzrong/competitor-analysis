# 正确云 | 深度分析报告

**分析日期**：2026-05-23
**分析范围**：产品功能、用户体验/交互、目标用户与定位、技术能力四维度全覆盖
**分析背景**：正确云是北京正确教育旗下的一站式备授课平台，与同公司旗下的橡皮网（题库组卷）、指课网（教师培训）、正确阅卷（阅卷分析）形成完整产品矩阵，与学科网产品线高度对应。

---

## 概况

- 产品名称：正确云
- 所属公司：北京正确教育咨询有限公司
- URL：https://www.zhengqueyun.com / https://www.zqy.com
- 产品类型：🔴核心直接竞品（K12备授课平台）

---

## 一、产品功能分析

### 核心功能

正确云定位为"一站式备授课工具"，功能覆盖备课、授课、协作全流程：

| 功能模块 | 实现方式 | 与学科网的重叠度 |
|---------|---------|----------------|
| 备课空间 | 个人云存储，数据加密，30天可恢复误删资源 | 高（学科网云备课功能） |
| 海量备课资源 | 按章节精准匹配整册教材资源 | 高 |
| 在线授课 | 无需下载安装，空间资源一键授课 | 中（学科网以资源下载为主） |
| 集体备课 | 多人协作，任务发布、资源共享 | 高（学科网集体备课功能） |
| 校本空间 | 学校自主建设知识体系，自定义管理教材与知识点 | 高（学科网校本资源库） |
| 试题/试卷/课件/教案/学案/视频 | 全覆盖 | 高 |

### 独有功能（学科网未覆盖或较弱）

1. **校本空间** — 关注度：**中**
   - 学校自主建设知识体系，自定义管理教材与知识点。
   - 与学科网的校本资源库功能相似，但正确云强调"自定义管理"的灵活性。

2. **集体备课协作** — 关注度：**中**
   - 多人协作备课，支持任务发布和资源共享。
   - 学科网也有集体备课功能，但正确云的协作流程设计可能更成熟。

3. **在线授课（一键授课）** — 关注度：**中**
   - 无需下载安装，云资源直接用于课堂授课。
   - 学科网资源多以下载后本地使用为主，正确云的"一键授课"减少了下载步骤。

### 缺失功能（学科网有而正确云未覆盖）

- **AI能力**：正确云暂无公开AI能力，学科网已有AI小博士。
- **题库规模**：依赖橡皮网1500万+题库，远低于学科网2800万+。
- **组卷算法**：菁优网组卷体验行业最佳，正确云组卷功能一般。
- **品牌影响力**：学科网4万+学校 vs 正确云8000+学校。

### 功能更新频率

- 资源持续更新，但产品功能层面近年来无重大突破。
- 域名zhengqueyun.com到期（zqy.com为备用），存在品牌稳定性风险。

## 二、用户体验分析

### 关键流程

**备课流程：**

1. 登录正确云
2. 进入备课空间
3. 按章节筛选资源
4. 在线编辑或直接一键授课
5. 集体备课时邀请同事协作

### 体验评价

**亮点：**

- **云存储便利性**：备课资源云端存储，多设备同步。
- **一键授课**：减少下载-导入的繁琐步骤。
- **集体备课协作**：支持多人协作，适合教研组集体备课场景。
- **误删恢复**：30天内可恢复误删资源，降低操作风险。

**痛点：**

- **无AI辅助**：备课过程完全依赖人工搜索和筛选，效率低下。
- **资源体量不足**：与学科网相比，资源总量和质量有明显差距。
- **品牌认知度低**：相比学科网，品牌知名度和市场声量较小。
- **界面设计传统**：产品UI和功能设计多年未更新，体验落后。

### 可借鉴之处

1. **一键授课模式**：学科网可以探索"资源直链课堂"模式，减少教师的下载-导入操作。
2. **集体备课协作流程**：正确云的协作任务发布和资源共享机制可以作为学科网集体备课功能优化的参考。

## 三、定位与用户

- **目标用户**：中小学教师（覆盖小学、初中、高中全学段）
- **核心价值主张**："一站式备授课工具——云备课+在线授课+集体备课+校本空间"
- **定价模式**：学校网校会员制采购（与橡皮网共用账号体系）
- **市场切入点**：从学校集采切入，以"一站式备授课"打包方案获取客户
- **品牌调性**：老牌备授课平台，功能全面但缺乏亮点
- **用户口碑**：在部分学校有稳固客户，但整体口碑不如学科网

### 产品矩阵优势

正确云+橡皮网+指课网+正确阅卷的组合，在功能上与学科网的产品线高度对应：

| 正确教育矩阵 | 学科网对应产品 |
|-------------|--------------|
| 正确云（备授课） | 学科网备课+云资源 |
| 橡皮网（题库组卷） | 组卷网 |
| 指课网（教师培训） | 学科网教师研修 |
| 正确阅卷（阅卷分析） | 学科网阅卷 |

**关键判断**：正确教育矩阵是"小号版学科网"——产品线高度对应，但每个单品的体量和能力都弱于学科网。其威胁在于"打包销售"策略可能在一些预算敏感的学校获得青睐。

## 四、技术能力

- **已知技术栈**：传统备授课平台，技术栈无明显亮点
- **数据来源**：编辑团队整理+用户上传
- **AI能力**：暂无公开AI能力
- **技术亮点/壁垒**：无显著技术壁垒
- **技术局限性**：完全缺乏AI化能力，在大模型时代竞争力将持续下降
- **API开放能力**：未公开

---

## 五、总结与建议

### 核心差异

正确云是"小号学科网"——产品线高度对应，但在体量、AI能力、品牌影响力上全面落后。其最大威胁在于"打包销售"策略可能在部分学校分流学科网客户。

### 可借鉴点

1. **一键授课模式**：减少教师的下载-导入操作，提升使用便利性。
2. **集体备课协作流程**：任务发布和资源共享机制值得参考。

### 需关注的动态

1. **域名和品牌稳定性**：zhengqueyun.com域名到期，是否存在品牌迁移风险？
2. **AI化转型**：正确教育是否计划引入AI能力？若启动转型，竞争力将提升。
3. **打包销售策略**：是否在更多学校推广"正确云+橡皮网+指课网+正确阅卷"打包方案？
4. **外部资本介入**：正确教育是否会有外部投资加速产品升级？
