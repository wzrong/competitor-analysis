### ⭐ Transformer 联合作者 Noam Shazeer 离开 Google，加入 OpenAI

2017 年《Attention Is All You Need》联合作者、Google Gemini 联合负责人 Noam Shazeer 于 6 月 18 日宣布离开 Google 加入 OpenAI， reportedly 将负责 AI 架构研究。Google 不到两年前才通过约 27 亿美元的技术授权与人才交易从 Character.AI 将他召回，如今再次流失。这一人事变动被多家媒体称为 2026 年 AI 人才战争中最具标志性的跳槽之一。

**核心观点：** OpenAI 在 IPO 前夕密集招揽顶级研究与工程人才，Shazeer 的加入将直接增强其下一代模型架构能力。Google 在 48 小时内连失 Shazeer 与 Jumper 两位标志性人物，也加剧了外界对 Gemini 3.5 Pro 迟迟未全面发布背后研发节奏的担忧。

### ⭐ 诺奖得主 John Jumper 离开 DeepMind，加入 Anthropic

**核心观点：** AI 行业的人才争夺已进入「诺奖级」阶段。Anthropic 在 IPO 前夕从 Google 连挖两位顶级人才，显示其有意抢占科研高地，并可能在 AI for Science 方向展开重大布局。

### ⭐ DeepSeek 全力押注 Agent Harness，负责人开启「Boss 直聘」模式

DeepSeek Harness 部门负责人崔添翼近一个月持续在 X 上高调招人，最新放出 Harness 研究员、Harness 工程师、Harness 产品经理三类岗位。他在采访中表示「每天都在面试，以及各种地方贴小广告」，但招了一个多月仍未招够。DeepSeek 对 Harness 的定义是：如果 Agent 是汽车、模型是发动机，Harness 就是方向盘、变速箱与刹车——即模型之外让 Agent 可用、可控、可进化的全套工程层。

**核心观点：** 大模型上半场抢的是研究员，下半场 Agent 阶段抢的是能把研究、工程、产品与用户需求揉到一块的复合型团队。DeepSeek 的招聘 JD 几乎可以当作一份 Agent 能力坐标图：Context Engineering、长期记忆、Subagent / Multi-Agent、Tool Use、Planning、MCP 等。

来源：[量子位](https://www.qbitai.com/2026/06/437249.html)

### ⭐ OpenAI o3 辅助诊断 18 例罕见儿童遗传病

OpenAI 与波士顿儿童医院、哈佛大学合作的研究于 6 月 18 日发表在 NEJM AI 上。研究使用 OpenAI o3 Deep Research 推理模型分析了 376 例此前未确诊的病例，最终帮助医生确诊了 18 例（额外诊断率 4.8%）。同日，OpenAI 还将健康智能更新部署到 GPT-5.5 Instant，使免费用户也能获得临床级健康指导。目前每周有超过 2.3 亿人使用 ChatGPT 进行健康咨询。

**核心观点：** 这是 AI 在临床医学中最具说服力的应用案例之一——帮助解决专家也无法解决的罕见病诊断。将健康智能同步推向免费用户，显示 OpenAI 在医疗领域的长期战略野心。

来源：[OpenAI 官方](https://openai.com/index/diagnose-rare-childhood-diseases/) | [NBC New York](https://www.nbcnewyork.com/news/health/ai-boston-childrens-hospital-diagnose-rare-diseases-kids-openai/6515476/)

### ⭐ 智谱 GLM-5.2 支持 1M 上下文，下周 MIT 开源

智谱于 6 月 13 日面向 GLM Coding Plan 全量用户开放 GLM-5.2，并宣布下周遵循 MIT 协议开源权重。GLM-5.2 定位面向 AI Coding / Agentic Engineering，支持 100 万 token 上下文窗口，最大输出可达 131,072 tokens，提供 High / Max 两种思考模式。同日发布的 ZCode 3.0 全面切换自研 ZCode Agent 内核，深度适配 GLM-5.2，GLM Coding Plan 用户可享受 150% 配额。

**核心观点：** 国产开源模型正在从「追赶 benchmark」转向「围绕 Coding Agent 场景做端到端优化」。1M 上下文 + MIT 开源 + 自研 Agent 工具链，意味着智谱试图在开发者生态中建立新的默认选择。

来源：[Z.ai Release Notes](https://docs.z.ai/release-notes/new-released) | [AIBetas](https://www.aibetas.com.cn/p/6274.html)
