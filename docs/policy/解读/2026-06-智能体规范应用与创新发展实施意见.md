## 二、核心内容摘要

1. **首次国家级定义 AI Agent**：明确"智能体"是具备目标理解、任务规划、工具调用、结果反馈四要素的 AI 系统，与单次问答式 AI 区分
2. **鼓励教育垂类 Agent 开发**：教育被列为智能体优先应用赛道之一，与政务、医疗、制造并列
3. **Agent 备案**：面向公众提供服务的智能体需完成算法备案 + 智能体应用备案
4. **内容审查**：智能体输出内容纳入生成式 AI 内容审查框架
5. **分级分类监管**：按智能体的自主性、决策权限、影响范围分级监管；高危智能体（涉及自动决策、金融、医疗等）需前置审批
6. **教育场景专项引导**：明确鼓励备课 Agent、教研 Agent、评价 Agent 开发，暂缓自主决策型 Agent（如自主判定学生等级、自动生成评价）

## 三、对学科网的影响

### 学科网自身产品

| 产品/规划中形态 | 分级 | 监管要求 | 应对 |
|---|---|---|---|
| AI 小博士（AI 备课 Agent 形态）| 低危 | 需算法备案 + 智能体应用备案 | 3 个月内完成双备案 |
| AI 研修（教师成长 Agent 形态）| 低危 | 需备案 | 3 个月内完成备案 |
| 未来 · 教研 Agent | 低危 | 备案 | 立项前评估 |
| 未来 · 学情评价 Agent（若含自主判定学生等级）| **中/高危** | **前置审批** | **立项前评估自主性边界，避免踩线** |
| 未来 · 教学决策 Agent（若含自动生成教学方案）| **中危** | **前置审批** | **保留教师最终决策权，避免自主性升级到中危** |

### 对竞品的差异化影响

| 竞品 | 影响 |
|---|---|
| 好未来九章爱学（CEO 明确 AI Agent 核心战略）| 🟡 战略正名，但备案成本上升；与实施意见时间点吻合 |
| 猿辅导飞象老师（"课堂运行系统"= Agent 化）| 🟡 战略正名，飞象 2.0 需完成备案 |
| 智学网（讯飞 AI 学习机 Agent 化）| 🟡 战略正名，硬件 Agent 需备案 |
| 字节-豆包老师/豆包爱学 | ⚠️ 高危场景（拟人化+自主性）叠加，合规成本高 |
| 钉钉教育（AI Agent 背景 CEO 陈宇森）| 🟡 战略方向正确，合规成本可接受 |
| 智谱 AI 拍学机（教育 Agent 硬件）| 🟡 战略正名 |

**关键判断**：实施意见对 Tier 1（核心战略竞品）竞品**整体是"政策利好+合规成本"双面效应**。战略上确认了 AI Agent 是教育信息化下一阶段的合法方向；执行上抬高了备案门槛。**对学科网是"必须做且能做"的方向 —— 现在不做 Agent，2 年内会失去政策定义的下一代教育产品身份**。

## 四、资金流向判断

- **短期（3-6 个月）**：政府采购目录可能新增"教育智能体"类目；教科信〔2026〕1号已经激活的各省 Q2-Q3 采购预算中，Agent 类产品成为热门
- **中期（1 年）**：教育 Agent 采购规模化，估算 2027 年 B 端教育 Agent 市场规模 20-50 亿元
- **对学科网**：机会窗口 —— AI 小博士升级到 Agent 形态、AI 研修 Agent 化可以承接采购需求

## 五、建议行动

| 受众 | 行动 |
|---|---|
| **产品线** | ①启动 AI 小博士、AI 研修的算法备案+智能体应用备案（3 个月内）；②规划中的评价 Agent、教学决策 Agent 立项时前置评估自主性边界；③把"教育 Agent"作为下一代旗舰产品线定位 |
| **市场线** | ①在教育信息化项目招标中，主动提"我们已完成教育智能体备案"作为差异化优势；②话术卡新增"符合国家智能体规范"专项段落 |
| **决策层** | ①明确 Agent 化是下一代战略方向；②备案成本纳入产品预算；③评估是否设立"教育智能体研究团队"承接备案+研发一体化 |
