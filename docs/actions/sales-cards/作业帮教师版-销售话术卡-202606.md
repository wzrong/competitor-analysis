# 作业帮教师版 销售话术卡

**更新日期：2026-06-01**

---

## 他们的进攻话术（已知）

1. 「拍照就能出题，手机一扫自动生成试卷，方便」
2. 「基础功能免费，老师个人用不花钱」
3. 「亿级学生的错题数据，变式训练更精准」
4. 「作业帮学生都在用，家长老师都熟悉这个品牌」

---

## 我们的反制

| 他们说 | 我们回 |
|---|---|
| 「拍照出题，方便」 | 拍照识题确实方便，我们认可这个体验。但用于学校正式考试，题目来源和版权是第一道门槛——C 端用户上传的题目来源杂，拿来出正式考试卷有风险。我们 3000 万道题，每一道都有来源、难度标注、课标对应，出正式卷放心。 |
| 「基础功能免费」 | 免费的是个人用的那部分。学校采购的是整体方案——阅卷、学情分析、集体备课、教师研修，这些都不在免费范围里，而且作业帮教师版目前也没有这些功能。学校级需求，免费工具解决不了。 |
| 「亿级数据，推题精准」 | 亿级学生数据做变式推题确实是优势。但学校老师需要的不只是推题，还需要出完整的单元测验卷、批改、出学情报告——这个链路在作业帮里是断的，老师还得另找工具。 |
| 「品牌知名」 | 学生端知名，教师端和学校决策层知名度有限。学科网 22 年深耕学校市场，4.5 万+合作学校里的教务主任和校长更熟悉我们。采购决策看的是学校级口碑，不是学生端品牌。 |

---

## 引导问题（让客户自己发现对方弱点）

- 「老师用作业帮出完题，可以在平台上直接批改、自动统计错题率吗？」
- 「学校要给教师做研修、记学时，作业帮有这个体系支持吗？」
- 「正式期末考试的试卷，用拍照上传的题目，有没有版权方面的顾虑？」
- 「教研组集体备课，多个老师协同修改同一份教案，作业帮支持吗？」

---

## 绝对不要说的话

- ❌「拍照出题不可靠」（拍照识题技术是真实有效的，否认会失去可信度）
- ❌「作业帮是 C 端学生工具，不懂学校」（它确实在做教师工具，轻视会被打脸）
- ❌「免费就是没价值」（免费是商业策略，不是质量问题）
- ❌「亿级数据没用」（数据优势是真实的，特别是推题精准度方面）
