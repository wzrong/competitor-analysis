# 九章爱学 · 销售话术卡

**更新日期：2026-06-01**

---

## 他们的进攻话术（已知）

1. 「我们的MathGPT数学AI是全球领先的，能讲解解题过程」
2. 「170多个智能体，老师教学全周期都覆盖了」
3. 「好未来做了30年教育，AI是真的懂教育的」
4. 「我们的1平台+3块屏，备课授课一套全搞定」
5. 「学生端小精龙和老师版联动，学情数据全链路打通」

---

## 我们的反制

| 他们说 | 我们回 |
|---|---|
| 数学AI全球领先 | 数学这块他们确实做得深，但教学不只有数学——我们覆盖全学科2700万+资源，语文、英语、理化生，哪个学科的老师都能用好，不是只为数学老师服务的平台。 |
| 170个智能体覆盖全周期 | 智能体多不等于实际能用，老师真正高频使用的功能有几个？学科网的核心产品经过22年迭代，4.5万+学校验证，是真实使用中打磨出来的深度。 |
| 好未来懂教育 | 好未来当然懂教育，但他们过去是做学科培训起家的，K12学校服务是近几年才重点发力的。学科网从2004年开始就专做学校端，在学校场景的理解和产品深度上有更长的积累。 |
| 1平台+3块屏全搞定 | 3块屏的方案听起来很完整，但硬件采购谁来负责？软硬件整合谁来维护？学科网是纯软件订阅，学校已有的硬件都能用，没有额外硬件采购压力。 |
| 学情数据全链路打通 | 学生端数据联动确实是亮点。但这个链路要成立，需要家长和学生也装小精龙——在学校场景推动家长安装新App的难度不低，实际落地效果值得观察。 |

## 引导问题

- 「老师在语文/英语课备课的时候用九章爱学，资源够用吗？」
- 「他们的组卷和阅卷功能有吗？考试之后学情怎么处理的？」
- 「1平台+3块屏，硬件部分是谁采购？和他们的平台绑定的吗？」
- 「170个智能体，您们学校的老师实际在用的有几个？」
- 「小精龙需要学生单独下载吗？家长配合度怎么样？」

---

## 绝对不要说的话

- ❌「好未来快完了/双减之后就不行了」（FY2026年收入30亿美元，说了被当场打脸）
- ❌「MathGPT没那么厉害」（数学AI确实领先，否认会失去可信度）
- ❌「我们的AI比九章强」（在数学场景明显不成立）
