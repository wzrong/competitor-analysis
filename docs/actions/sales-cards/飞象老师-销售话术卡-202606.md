# 飞象老师 · 销售话术卡

**更新日期：2026-06-01**

---

## 他们的进攻话术（已知）

1. 「我们的AI课件生成速度更快，一份教案就能出一整节互动课」
2. 「我们有课堂数据回收，学生在微信上答题，老师实时看到情况」
3. 「我们是免费的，学科网要收费」
4. 「我们有3D动画、游戏化课件，学生更喜欢上课了」
5. 「我们在世界数字教育大会上亮相，是教育部认可的平台」

---

## 我们的反制

| 他们说           | 我们回                                                                         |
| ------------- | --------------------------------------------------------------------------- |
| AI课件生成更快      | 快是因为AI生成，但准不准是另一回事。学科网的资源经过教研审校，老师用了不需要再检查课标对不对，省的时间比生成快那几分钟要多得多。           |
| 课堂数据回收是亮点     | 这确实是好功能，但学校选平台是选一整套——我们覆盖备课+组卷+阅卷+学情+研修全链路，他们只有备课+课中，剩下的还得额外买其他系统，算下来总成本不低。 |
| 我们是免费的        | 免费是现阶段的获客策略，积分用完就要付费。学科网是透明的年费制，学校能预期全年成本，不会遇到突然调价的风险。                      |
| 3D动画/游戏化学生更喜欢 | 有趣的课件能提高学生参与感，这个认可。但课堂效果的核心是内容准确性——AI生成的内容有幻觉风险，我们的内容经过教研团队多轮审校，老师用起来放心。    |
| 教育部认可的平台      | 世数会有很多企业参展，亮相不等于独家认可。学科网也在多个国家级教育活动中参与，更重要的是服务了4.5万+学校、5700万+注册用户的实际验证。     |

---

## 引导问题（让客户自己发现对方弱点）

- 「老师们在用飞象老师的时候，出来的课件需要自己再检查一遍吗？课标方向有没有跑偏过？」
- 「他们的平台有组卷功能吗？考试完了之后学情分析怎么做的？」
- 「老师有在用他们的教师研修功能吗？」
- 「飞象老师的积分包大概多少钱一个？用量大的老师每个月要花多少？」
- 「学校如果要上他们的阅卷，要再买别的系统吗？」

---

## 绝对不要说的话

- ❌「飞象老师那个课件是假的/做不到」
- ❌「他们的AI一塌糊涂」（当前不成立）
- ❌「我们的AI比他们强」（会被当场对比拆穿）
- ❌「猿辅导要倒了」（无根据，失去专业形象）
