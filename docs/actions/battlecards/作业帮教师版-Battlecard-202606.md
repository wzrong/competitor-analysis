## 核心差异（客观）

**✅ 我们更强的3条**

1. **B 端渠道和学校关系远强于作业帮**：学科网 22 年深耕 B 端，4.5 万+合作学校、完整的学校级采购和服务体系；作业帮教师版以 C 端 App 为主，学校级采购关系尚在建设中，B 端渠道体系远不成熟。
2. **功能完整度压制**：学科网覆盖备课→组卷→阅卷→学情→教师研修全链路；作业帮教师版核心是拍照组卷+变式训练，阅卷、教师研修、集体备课等功能缺失，是移动端工具而非教学平台。
3. **内容质量和版权保障**：学科网 2700 万+资源均经过教研审校、版权清晰；作业帮题库来源于 C 端用户拍照上传，版权和内容准确性有一定风险，不适合直接用于正式考试出题。

**❌ 他们更强的（客观承认）**

1. 拍照组卷独特且便捷：用手机拍一道题→自动匹配相似题→生成试卷，是学科网没有的移动端组卷方式，对习惯手机操作的教师有真实吸引力
2. C 端亿级学生数据反哺：错题大数据可以使「个性化推题」更精准，这是学科网 B 端模式无法复制的底层优势
3. 免费基础功能拉低使用门槛：拍照搜题、组卷、变式训练基础版免费，教师个人使用零成本

## 常见对比场景

**当教师说「作业帮教师版免费，学科网要收费」**
→ 免费的是基础功能，学校级采购方案同样需要付费。更关键的是：免费工具解决不了学校的系统需求——阅卷、研修、集体备课、学情档案，这些免费工具都没有。学校采购决策看的是整体价值，不是单功能免费。

**当教师说「拍照出题很方便，手机就能用」**
→ 拍照识题是个好功能，我们承认便捷。但出卷用于正式考试，题目版权和准确性是前提——C 端用户上传的题目来源杂，不适合直接用于学校正式考试。学科网 3000 万道题都有明确版权和审校，出正式考试卷更安全。
