# 市场监测 · 索引

> 资金流向：钱往哪里流。跟踪招投标、采购与区域市场信号，用于判断机会窗口。

> 模块状态：🟡 试运行中，Phase 2 首轮外部招投标搜索已完成

## 模块定位

市场监测回答「钱往哪里流」。当前先聚焦教育信息化、AI 教育、资源平台、备授课、测评阅卷相关招投标信号，用来支撑三类判断：

1. 区域采购热度：哪些省市正在释放预算和项目机会。
2. 竞品商业落地：Tier 1/2 竞品是否出现中标、续约、区域集采或学校采购。
3. 新进入者预警：陌生企业是否通过采购记录进入教育数字化场景。
