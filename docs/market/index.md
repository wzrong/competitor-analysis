# 市场监测 · 索引

> 模块状态：🟡 试运行中，Phase 2 首轮外部招投标搜索已完成

## 模块定位

市场监测回答「钱往哪里流」。当前先聚焦教育信息化、AI 教育、资源平台、备授课、测评阅卷相关招投标信号，用来支撑三类判断：

1. 区域采购热度：哪些省市正在释放预算和项目机会。
2. 竞品商业落地：Tier 1/2 竞品是否出现中标、续约、区域集采或学校采购。
3. 新进入者预警：陌生企业是否通过采购记录进入教育数字化场景。

## 联动出口

市场监测产出后必须检查是否触发联动中心：

| 信号 | 去向 | 动作 |
|---|---|---|
| Tier 1/2 竞品中标、续约、区域集采 | 竞争分析 | 建议更新竞品 L3/L4/L5，必要时进入发版/深度分析队列 |
| 新进入者单笔金额显著或多次中标 | 竞争分析 | 建议纳入观察池或新建档案 |
| 某省市教育数字化采购密集出现 | 政策分析/行业分析 | 检查政策、财政或专项行动背景 |
| AI 备课、测评、资源平台采购品类升温 | 行业分析/应对建议 | 纳入趋势信号，必要时更新销售话术或 Battlecard 候选 |
