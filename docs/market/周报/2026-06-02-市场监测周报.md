## 八、A 级证据联动分析

### 8.1 飞象星球北京多项目中标

| 分析维度 | 结论 |
|---|---|
| 事实边界 | A 级证据来自北京市公共资源交易服务平台，明确供应商为北京飞象星球科技有限公司；3 个项目公告时间分别为 2025-05-20、2025-07-04、2026-03-25，合计约 636.04 万元，覆盖市级数字教育中心、平谷区教委、东城区智慧教育研究中心。 |
| 采购内容 | 不是单一“飞象老师”标准化产品采购，而是 AI 通识课程、智慧作业、AI 作文、数字学校资源建设等项目制服务。 |
| 竞争含义 | 飞象星球已经具备北京教育系统项目制交付能力，且能把 AI 内容、作业、作文、课程资源打包进政府采购；这会强化猿辅导从 C 端/工具型品牌向 B 端项目型方案商转化的可信度。 |
| 对学科网威胁 | 威胁不在单点功能，而在“AI 课程+作业服务+区域项目”的组合进入同一批教育局预算池；如果飞象后续把教师备课/资源生成能力嵌入项目，将直接挤压学科网资源平台与 AI 备课工具的 B 端采购空间。 |
| 应进入模块 | 竞争分析 P0：更新猿辅导-飞象老师 L3 商业化、L4 渠道力、L5 组织/交付能力；市场监测继续追北京区级项目。 |
| 下一步动作 | 优先追龙岗区 430.3 万元待核实项目原始公告；下一次猿辅导-飞象老师档案复核时，把“北京项目制落地”作为独立渠道信号处理。 |

### 8.2 希沃智慧黑板采购

| 分析维度 | 结论 |
|---|---|
| 事实边界 | A 级证据来自甘肃经济信息网，公告时间为 2026-05-15；成交供应商为甘肃新赶考教育科技有限公司，希沃/广州视睿出现在采购清单的品牌、生产厂家和软件项中。 |
| 采购内容 | 采购清单包含 86 寸智慧黑板、设备统一管理软件、希沃白板 5、无线传屏系统、视频展台，成交金额 23.49 万元。 |
| 竞争含义 | 这不是“希沃主体中标”，而是“经销商/集成商中标 + 希沃产品进入学校采购清单”。它说明希沃仍通过硬件渠道稳定进入学校，同时把备授课软件和设备运维软件自然捆绑进项目。 |
| 对学科网威胁 | 希沃的威胁在入口控制：硬件一旦进校，希沃白板、设备运维和课堂工具会成为教师日常使用环境，削弱独立资源/备课平台的进入频次。 |
| 应进入模块 | 竞争分析 P1：希沃 L4 渠道力补充“经销商承接学校项目，希沃以品牌/软件清单进入”；行业分析 P1：跟踪硬件采购中软件项占比。 |
| 下一步动作 | 后续市场监测记录希沃项目时，区分“希沃主体中标”和“希沃产品被集成商采购”；不要把所有希沃清单项目等同为希沃直接中标。 |

### 8.3 菁优网学校单一来源采购

| 分析维度 | 结论 |
|---|---|
| 事实边界 | A 级证据来自新疆师范大学资产管理处，公示时间为 2023-09-25；采购人为新疆师范大学附属中学，项目为菁优网采购，预算 75,080 元，拟定供应商为深圳市菁优智慧教育股份有限公司。 |
| 采购内容 | 单一来源理由强调题库覆盖小初高、多教材版本、题量、组卷、下载、在线训练和进阶推题能力。 |
| 竞争含义 | 菁优网的学校采购逻辑很清楚：不是区域大项目，而是围绕题库、组卷、训练形成学校级刚需，金额小但可复购。 |
| 对学科网威胁 | 菁优网对学科网的威胁集中在理科题库与组卷效率。如果学校以单一来源方式认可其不可替代能力，说明其在特定学科/题库场景具备采购理由，而不只是教师个人付费工具。 |
| 应进入模块 | 竞争分析 P1：更新菁优网 L3 收入模型和 L4 学校采购路径；市场监测保留“学校低金额年费采购”样本池。 |
| 下一步动作 | 继续追踪是否有同类学校续费、区县资源平台入围或教育局级采购；与橡皮网、学科网同校采购情况一起做资源平台替代关系分析。 |

### 8.4 橡皮网学校单一来源采购

| 分析维度 | 结论 |
|---|---|
| 事实边界 | A 级证据来自新疆师范大学资产管理处，公示时间为 2023-09-25；采购人为新疆师范大学附属中学，项目为橡皮网采购，预算 56,000 元，拟定供应商为北京正确教育咨询有限公司。 |
| 采购内容 | 单一来源理由强调 K12 题库、原创/名校资料、按课时和知识点组卷、视频专区、教研培训内容等。 |
| 竞争含义 | 橡皮网不是只卖题库，而是在学校端包装为“题库+资料+视频教研”的综合资源服务；这使它与学科网的资源、组卷、教研内容存在更直接重叠。 |
| 对学科网威胁 | 威胁不是金额规模，而是同一所学校可以同时采购多个资源型竞品，说明学校资源预算可能被拆分到多个平台；学科网需要关注“单校多平台订阅”下的替代与互补关系。 |
| 应进入模块 | 竞争分析 P1：更新橡皮网 L3 收入模型和 L4 学校采购路径；正确云仅保留公司级侧证，不得写成正确云独立采购。 |
| 下一步动作 | 继续追踪橡皮网是否出现教育局/区域采购；同时把“橡皮网采购不能等同正确云采购”的证据边界写入后续复核口径。 |
