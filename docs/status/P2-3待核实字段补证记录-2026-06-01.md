# P2-3 待核实字段补证记录 | 2026-06-01

> 范围：猿辅导-飞象老师、作业帮教师版、好未来-九章爱学
> 原则：只升级有公开来源或本地已沉淀可追溯来源支撑的字段；价格、校采单价、招投标、用户规模等硬字段没有证据时继续保留「待核实」。

## 一、本轮结论

本轮不是一次性补完 P2-3，而是完成第一批字段边界清理：

- 猿辅导-飞象老师：修正飞象星球整体口径与飞象老师单品口径的混用，补强 B 端产品矩阵、服务规模和硬件/空间方案边界。
- 作业帮教师版：未找到足以支撑教师版独立用户规模、B 端定价、硬件捆绑关系的公开证据，保留待核实，仅把「集团层面线索不能等同教师版事实」写入档案边界。
- 好未来-九章爱学：补强「1平台+3块屏」全场景方案、仁北高中联合实验室和硬件合作伙伴待核实边界。

## 四、对后续补证的要求

1. 价格字段只接受公开价格页、购买页、App 内购买截图或招投标/采购公告。
2. B 端采购字段优先搜索中标公告、政府采购网、教育局采购公示、学校采购合同。
3. 用户规模字段必须区分集团、产品矩阵、单品、教师端、学生端，不做口径合并。
4. 硬件字段必须区分「方案中包含硬件」和「与某硬件厂商捆绑销售」。
